Components of a Computer in the CPU

  • Central Processing Unit (CPU)

    The CPU is the brain of the computer, responsible for executing instructions and performing calculations. It is made up of the ALU, CU, and Registers.

  • Arithmetic Logic Unit (ALU)

    The ALU performs mathematical operations, such as addition and subtraction, and logical operations, such as AND and OR.

  • Control Unit (CU)

    The CU manages the operations of the CPU, including fetching instructions, decoding them, and coordinating the activity of the ALU and Registers.

  • Registers

    Registers are small, high-speed storage units within the CPU. They hold data that is being processed or data that is waiting to be processed.

  • Cache Memory

    Cache Memory is a small, fast memory system within the CPU that stores frequently accessed data for quick access. This helps to speed up the processing of instructions.
